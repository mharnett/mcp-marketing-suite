# Contributing

This repo covers the suite README, install script, and per-platform docs. For code changes to individual MCPs, submit PRs to their repos:

- [mcp-google-ads](https://github.com/mharnett/mcp-google-ads)
- [mcp-bing-ads](https://github.com/mharnett/mcp-bing-ads)
- [mcp-linkedin-ads](https://github.com/mharnett/mcp-linkedin-ads)
- [mcp-ga4](https://github.com/mharnett/mcp-ga4)
- [mcp-google-gsc](https://github.com/mharnett/mcp-google-gsc)
- [meta-ads-mcp](https://github.com/mharnett/meta-ads-mcp)

## For this repo

PRs welcome for:
- Corrections to the README or docs
- Improvements to the install script
- New workflow examples in `examples/`

Maintained by one person alongside client work. Expect 1–2 week review time. Keep PRs focused.

## Issues

Open issues on the individual MCP repos for bugs or feature requests specific to a platform. Open here for suite-level concerns (install script, docs, cross-MCP workflows).

Issues triaged within 7 days.
